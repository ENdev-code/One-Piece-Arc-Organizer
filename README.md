# One Piece Episode Organiser
**Coming Soon**
A python script that organises One Piece episodes according to arc and saga for both pre- and post-timeskip. Perfect for One Piece fans who want their episode collection to be as accurate as possible.

> Let's watch our favourite crews sail those seas in the best way we can

---

# Features

Below are the key features of this tool

| Feature                  | Description                                                                                                     |
|--------------------------|-----------------------------------------------------------------------------------------------------------------|
| Lore Accurate Organising | Episodes are neatly organised into their accurately named arcs from Romance Dawn until Egghead [current latest] |

# Use Cases

Below are some of the scenarios that may shine light on this tool's importance

| Use                      | Description                                                                                                   |
|--------------------------|---------------------------------------------------------------------------------------------------------------|
| Unorganised Episode list | if you have an unorganised episode list for One Piece but wanted a lore accurate organisation                 |
| Inaccurate Organisation  | Episodes may be organised but within folders that do not accurately portray the storyline and its arc titles  |
| Pride                    | Making your One Piece collection look more like a collectors stash.                                           |
